◆ calibrate()

void calibrate ( )

Calibrates the lifting motors (setscrews) using physical buttons.

Each motor has a dedicated button (CALIBRATION_BUTTON_1 for DXL_LIFT1 and CALIBRATION_BUTTON_2 for DXL_LIFT2). The motors move upwards until their respective button is pressed, at which point they stop. After both motors have been calibrated, they are set to their default positions.

◆ sequence()

void sequence ( )

Full sequence of moving the battery down, switching sides and moving it up again.

The sequence consists of the following steps:

  1. Move the battery down.
  2. Switch the lock state of the top side.
  3. Rotate to the opposite side.
  4. Switch the lock state of the new top side.
  5. Move the battery up.

void vTaskSetup ( void *  pvParameters)

Setup task that performs initial setup of motors and system.

This task sets up the motors, initializes the mutex for DXL communication, and starts the logging and main loop tasks. Once setup is complete, this task deletes itself.
