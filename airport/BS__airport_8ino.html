◆ setupMotors()

void setupMotors ( )

Sets up the DYNAMIXEL motors with the desired configurations. This includes:

  • Setting the profile acceleration and velocity for smooth movements
  • Configuring the motors to track absolute multi-turn positions
  • Setting the homing offset for the motors to ensure they start from a known position
  • Turning on the torque after configuration to enable motor movement The function iterates through all motors (both lift and rotation) to apply the necessary settings

◆ swapIndefinitely()

void swapIndefinitely ( )

Swaps the batteries indefinitely for testing purposes. The function waits for a button press to start the swapping sequence, and then continuously executes the sequence function in an infinite loop.

◆ TaskMotor()

void TaskMotor ( void *  pvParameters)

Acts as the only interface for communicating with the DYNAMIXEL motors. It listens for motor commands on a queue and executes them sequentially. The task runs indefinitely, processing commands as they come in. It also sends acknowledgments back to the requesting task if an acknowledgment task handle is provided in the command structure.

Parameters
pvParametersNo parameters are passed to this task.

◆ vTaskLogging()

void vTaskLogging ( void *  pvParameters)

Periodically reads motor data and logs it. This includes:

Parameters
pvParametersNo parameters are passed to this task.
  • Motor positions
  • Motor currents
  • Motor temperatures The task runs indefinitely with a delay between each logging cycle to avoid overwhelming the system with logs

◆ vTaskSetup()

void vTaskSetup ( void *  pvParameters)

Performs essential setup for the system, including:

Parameters
pvParametersNo parameters are passed to this task.
  • Setting up pins
  • Initializing serial communication
  • Setting up the DYNAMIXEL motors
  • Calibrating the system if not in manual mode After setup is complete, it starts the main loop, logging, and watchdog tasks, and then deletes itself as it is no longer needed.

◆ vTaskWatchdog()

void vTaskWatchdog ( void *  parameter)

Watchdog task that periodically performs a health-check.

This task pings all the motors and halts the system if any of them doesn't respond.

Parameters
pvParametersPointer to task parameters (not used).
